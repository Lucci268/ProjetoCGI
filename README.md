# 🎨 Projeto de CGI

## 📌 Descrição

Este projeto de **Computação Gráfica Interativa (CGI)** tem como objetivo a modelagem e animação de uma entidade visual inspirada no universo Pokémon: o **Spiritomb**.

A escolha do personagem se deve à sua composição singular e ao seu potencial expressivo para simulações gráficas dinâmicas.

## 🧩 Estrutura da Figura

O modelo desenvolvido é composto por três elementos principais:

- **Pedra (Base/Corpo)**  
  Representa a estrutura física que ancora a entidade.

- **Névoa (Cabeça)**  
  Elemento fluido que simula a essência espectral do personagem.

- **Rosto (Olhos e Boca)**  
  Componentes responsáveis pela expressividade da figura.

## 🔄 Animação e Interatividade

A figura apresenta:

- Movimento de **rotação contínua**, criando um efeito dinâmico;
- **Interação entre os componentes**, reforçando a coesão visual do modelo;
- Simulação de comportamento orgânico entre as partes.

## ✨ Efeitos Visuais

Para aumentar o nível de realismo e fidelidade visual, foram implementados:

- Sistema de **partículas**, simulando energia espectral;
- Movimentação fluida da névoa;
- Uso de **biblioteca externa** para suporte na renderização e controle das partículas.

## 🛠️ Tecnologias Utilizadas

- Biblioteca gráfica externa para manipulação de partículas  
- Técnicas de animação em CGI  
- Modelagem de objetos compostos  

## 🎯 Objetivo

Explorar conceitos fundamentais de CGI, incluindo:

- Modelagem de objetos compostos  
- Animação e transformação espacial  
- Uso de partículas para efeitos visuais  
- Integração de bibliotecas externas
